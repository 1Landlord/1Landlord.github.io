    Сокращения

    Свойство background

    background: [bc] [bi] [br] [bp] [ba];
    Обозначения:
    [bc] — background-color
    [bi] — background-image
    [br] — background-repeat
    [bp] — background-position
    [ba] — background-attachment
    Пример:background: #7f8c8c url("avatar.png") no-repeat 50% 50%;без [ba]
    Смещение тени

    box-shadow:
    [inset] - внутренняя
    +- 5px - смещение по x
    +- 5px - смещение по y
    [2px] - размытие
    [3px] - растяжение
    [red]; - цвет
    размытие есть, растяжение отрицательное — тень очень светлая;
    размытие есть, растяжения нет — тень обычная;
    размытие больше растяжения — тень темнее обычного;
    размытие меньше растяжения — тень очень темная.
    Параметры перехода перечисляются через пробел:

    свойство, длительность, форма и задержка:
    Пример:transition: width 1s ease-in 2s;
